Tujuan
Menetapkan PIC operasional utama dan cadangan untuk memastikan kelangsungan operasional sistem SIAP PMI pada lingkup satu provinsi sesuai prinsip governance dan change freeze.

Tanggung Jawab
1. Menjaga kepatuhan prinsip event-based & role per event.
2. Memastikan proses kelulusan melalui rapat manusia; sistem hanya mencatat.
3. Menegakkan bahwa evaluasi/KAP tidak mempengaruhi kelulusan.
4. Menjaga change freeze; mengelola bugfix melalui prosedur change control.
5. Menyusun dan menjaga evidence pack untuk audit internal.

Jalur Eskalasi Resmi
1. P0 (Kritis): Operator → PIC Utama → Pimpinan Unit. Target respon < 15 menit.
2. P1 (Tinggi): Operator → PIC Utama. Target respon < 1 jam.
3. P2 (Sedang/Rendah): Operator → PIC Cadangan/PIC Utama. Target respon < 24 jam.
